üì∏ Usage Guide

1. Authentication

Login: Use the toggle to switch between Candidate and Recruiter login screens.

Note: For this demo, any email/password combination will grant access.

2. Candidate Workflow

Select "My Profile Analysis" from the landing page.

Upload a resume (simulation) or click "Run AI Analysis".

View your personalized dashboard with skill gaps and job recommendations.

3. Recruiter Workflow

Select "Recruitment Hub" from the landing page.

Enter a Job Title and Description.

Click "Analyze & Find Candidates" to view the ranked list of applicants and talent heatmaps.
